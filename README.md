# 🚀 Skill Showcase Portfolio

An interactive, animated skills portfolio page built with pure HTML, CSS, and Vanilla JavaScript — no frameworks, no libraries.

Live demo of proficiency levels through scroll-triggered progress bars, circular SVG charts, a radar analysis canvas, and a dedicated projects section — all pulled from a real developer's CV.

---

## ✨ Features

- **Animated Progress Bars** — smooth left-to-right fill triggered on scroll via Intersection Observer
- **Circular / Radial Charts** — SVG stroke-dashoffset animation per skill
- **Radar Chart** — pure canvas radar covering all 12 skills, animated on scroll
- **Projects Section** — 4 real projects with tech stack badges and reveal animation
- **Category Tabs** — filter skills by Frontend, Backend, Database, Python, Tools
- **Hover Tooltips** — exact percentage + skill level description on hover
- **Dark / Light Mode** — toggle with localStorage persistence
- **Animated Counters** — stats strip with ease-out count-up animation
- **Fully Responsive** — mobile-friendly grid layout down to 320px
- **Keyboard Accessible** — arrow-key navigation on tab bar
